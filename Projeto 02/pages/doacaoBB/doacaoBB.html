Passo 1

1º -> Abra o aplicativo do banco do Brasil e acesse sua conta;
2ª -> Após o acesso da conta clique na opção transferências.

Passo 2

1ª -> No campo de Agência, preencha com o número da nossa agência 0000-0;
2ª -> No campo de Conta, preencha com o número da nossa conta 00000-0;
3º -> Vá na opção avançar.

Passo 3

1º -> No campo data pode escolher e preenchida com a data do dia, mas é possível o agendamento caso escolha uma data futura;
2º -> No campo valor, escolhe o valor desejado para doação;
3º -> Existe a opção repetir transferência, ao clicar no botão marcado de vermelho ativa essa opção, com isso você pode fazer o agendamento nos proximos meses;
4º -> Caso ativer repetir transferencia pode pular para o passo 5, se não vá para o passo 4.

Passo 4

1º -> O Campo marcado de vermelho você escolhe uma quantidade de meses para repetir;
2º -> O Campo marcado de verde você agendar até um mês escolhido;
3º -> O Campo marcado de azul a operação vai repetir até seu cancelamento.

Passo 5

1º -> Última etapa da transferência onde você informa sua senha de 6 digitos;
2º-> Por Último clicar em confirmar.

Passo 6

1º -> Casso ocorra tudo bem aparecerá uma imagem de sucesso e em seguida avançará para a tela acima onde poderá enviar o comprovante;
2º -> Casso ocorra um erro veja o erro e tente refazer a parte do erro ou entre em contato com a gente.
